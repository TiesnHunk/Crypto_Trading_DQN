Data storage cho project - Multi-Coin & Single-Coin datasets.

## 📂 Raw Data

### 🌟 multi_coin_1h.csv (Primary Dataset)
**Multi-cryptocurrency training data** - **USE THIS!**

**Stats:**
- **173,683 rows** (2.5x more than single-coin)
- **5 cryptocurrencies**: BTC, ETH, BNB, SOL, ADA
- **22 features**: OHLCV + 17 technical indicators
- **Timeframe**: 1 hour candles
- **Period**: 2016-2025

**Columns:**
```
timestamp, open, high, low, close, volume, coin,
rsi, macd, macd_signal, macd_hist,
sma_20, sma_50, ema_12, ema_26,
bb_upper, bb_middle, bb_lower, adx,
trend, price_change, volatility
```

**Coin Distribution:**
- BNB: 69,391 rows (40%)
- BTC: 68,518 rows (39%)
- ETH: 33,101 rows (19%)
- SOL: 1,368 rows (0.8%)
- ADA: 1,305 rows (0.8%)

**Source:** Kaggle datasets via `kagglehub`

**Generated by:**
```bash
python src/data/prepare_multi_coin_data.py --timeframe 1h
```

## 🚀 Usage

### Load Multi-Coin Data

```python
import pandas as pd

# Load multi-coin dataset
df = pd.read_csv('data/raw/multi_coin_1h.csv')

# Filter by coin
btc_data = df[df['coin'] == 'BTC']
eth_data = df[df['coin'] == 'ETH']

# Check available coins
print(df['coin'].unique())  # ['BTC', 'ETH', 'BNB', 'SOL', 'ADA']
```

### Load Trained Model

```python
import pickle

# Load latest Q-table
with open('src/checkpoints/checkpoint_latest.pkl', 'rb') as f:
    checkpoint = pickle.load(f)
    Q = checkpoint['Q']
    metadata = checkpoint['metadata']
```

### Prepare New Data

```bash
# Download all 5 coins from Kaggle
python src/data/prepare_multi_coin_data.py --download-only

# Prepare with indicators
python src/data/prepare_multi_coin_data.py --timeframe 1h

# Output: data/raw/multi_coin_1h.csv
```
