## 🚀 Usage

### View Charts

```python
from PIL import Image
import matplotlib.pyplot as plt

# Load and display
img = Image.open('results/charts/multi_coin_performance.png')
plt.imshow(img)
plt.axis('off')
plt.show()
```

### Load Backtest Results

```python
import pandas as pd

# Load trading history
df = pd.read_csv('results/reports/trading_history_replay.csv')

# Calculate metrics
total_trades = len(df)
profitable_trades = len(df[df['profit_pct'] > 0])
win_rate = profitable_trades / total_trades * 100

print(f"Total Trades: {total_trades}")
print(f"Win Rate: {win_rate:.2f}%")
```

### Generate New Charts
